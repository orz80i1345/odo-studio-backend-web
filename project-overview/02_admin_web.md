## 時段管理

功能：

- 設定營業時間
- 設定可預約時段
- 設定每個時段價格
- 設定是否啟用
- 設定公休日
- 設定維護日
- 設定包場日
- 設定特殊日期價格

時間格式限制：

資料庫不支援 TIME type，所以時段使用分鐘數。

範例：

| 顯示時間 | 儲存值 |
|---|---:|
| 00:00 | 0 |
| 09:00 | 540 |
| 14:00 | 840 |
| 19:00 | 1140 |
| 24:00 | 1440 |

14:00-19:00：

- `start_minute = 840`
- `end_minute = 1140`
- `duration_minutes = 300`

- 可以手動重新寄送密碼信

## Email 紀錄

可篩選：

- email_type
- status
- booking_id
- recipient_email

## 後台風格

- 清楚、耐看、適合長時間使用
- 可以比前台更工具化
- 仍使用同一套 OKLCH tokens
- 不要使用刺眼狀態色
